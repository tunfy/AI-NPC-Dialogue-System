# AI NPC Dialogue System

An intelligent NPC dialogue system built with Unity, FastAPI, SQLite and DeepSeek LLM.

---

## Project Overview

This project implements an AI-driven NPC dialogue system.

Players can interact with different NPCs in Unity, while the backend communicates with DeepSeek through FastAPI.

The NPC has:

- Independent personality
- Long-term memory
- Favorability system
- Persistent database

---
